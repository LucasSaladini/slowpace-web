# SlowPace 🌿

## 📌 Project Charter

### Título e Problema
**SlowPace.** Rastreadores de hábitos comuns utilizam gatilhos de pressão (*streaks*) e sobrecarga visual que geram ansiedade, transformando o lazer em obrigação. O projeto resolve isso através de uma interface acolhedora e sem punições por inatividade.

### Público-Alvo e Objetivo
Entusiastas de hobbies e pessoas neurodivergentes que buscam descompressão. O objetivo é criar um web app *low-stimulus* para registro de evolução pessoal, focado no prazer do processo e não em métricas de desempenho agressivas.

### Recursos e Ferramentas (Tech Stack)
* **Frontend/Backend:** Next.js (React), Node.js e Express (Hospedagem na Vercel).
* **Dados e Segurança:** PostgreSQL com Prisma ORM; Autenticação via JWT.
* **Gestão:** GitHub (Repositório) e GitHub Projects (Kanban).

### Métricas de Sucesso
* **Técnica:** Aplicação publicada na Vercel com integração funcional ao banco de dados.
* **Funcional:** Ciclo de cadastro e registro de atividade operacionais.
* **Incentivo:** Implementação de sistema de parabenização por tempo acumulado (recompensa não agressiva).
